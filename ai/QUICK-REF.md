# ⚡ QUICK REFERENCE - EMERGENCY CARD

> **For urgent situations when you don't have time to read full documentation**

## **🚨 BEFORE ANY CODE**
- [ ] **Check library docs first** - No custom solutions without research
- [ ] **Answer the 6 questions** in MANDATES.md self-assessment

## **📝 AFTER ANY SOLUTION**
- [ ] **Document immediately** - Update relevant .md files NOW
- [ ] **Run type-check** - `npm run type-check` before moving on

## **❓ BEFORE REMOVING CODE**
- [ ] **Ask user confirmation** - NEVER assume code is unnecessary

## **🔍 FOR ANY PROBLEM**
- [ ] **Find root cause** - Not just symptoms
- [ ] **Use problem-solving template** for complex issues

## **⚠️ RED FLAGS - STOP IMMEDIATELY**
- "I'll just implement this quickly..."
- "The user is waiting, I need to rush..."
- "I'll document this later..."
- "This should be simple..."
- "Time pressure means I can skip this step..."

## **🆘 WHEN IN DOUBT**
1. **STOP** what you're doing
2. **Read MANDATES.md** (30 seconds)
3. **Consult WORKFLOWS.md** for process
4. **Ask user** if still unclear

**Remember: Compliance is faster than fixing mistakes later**
